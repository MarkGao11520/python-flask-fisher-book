# 1.pipenv 按照与使用
#### [gitub地址](https://github.com/pypa/pipenv)
```
# 按照pipenv
pip install pipenv

# 进入虚拟环境（需要在项目目录上执行，如果当前目录没有，则会新建）
pipenv shell

# 退出虚拟环境
exit

# 使用pipenv按照类库
pipenv install #{package}

# 卸载类库
pipenv uninstall #{package}

# 查看按照包的依赖关系
pipenv graph

# 查看虚拟环境执行文件路径
pipenv --venv
```
