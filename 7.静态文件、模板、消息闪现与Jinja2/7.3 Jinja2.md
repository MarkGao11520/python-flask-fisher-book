# 7.3 Jinja2


### 1.jinja2模板渲染
读取字典和对象的两种方式

```
{# 注释 #}
{{ data.name}}
{{ data["age"] }}
```

条件控制语句
```python
{# 条件可以使用python的逻辑语言 #}
{% if data.age < 18 or data.name = '高文峰' %}
    aaa
{% elif data.age = 18 %}
    bbb
{% else %}
    ccc
{% endif %}
```
